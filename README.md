# Python for Data Science

A collection of Jupyter Notebooks covering the end-to-end data science workflow, from data manipulation to feature engineering. Datasets (`.csv`, `.xlsx`) are included alongside the notebooks for immediate reproducibility.

## 🗂️ Repository Contents

### 1. Numerical Computing (NumPy)
* Array and matrix operations.
* Vectorized operations and broadcasting.
* Real-world numerical simulations.

### 2. Data Wrangling (Pandas)
* **DataFrames & Series:** Indexing, filtering, and aggregation.
* **Data Manipulation:** `groupby`, `apply`, missing value handling.
* **Merging:** `concat`, `merge`, and joins.

### 3. Data Visualization (Matplotlib & Seaborn)
* Statistical data visualization.
* Distribution plots, scatter plots, and categorical plots.
* Plot customization and styling.

### 4. EDA & Feature Engineering
* Exploratory Data Analysis (EDA) workflows.
* Outlier detection and handling (e.g., Box plots).
* Categorical data encoding.
* Dataset balancing techniques.

## 🛠️ Tech Stack
* **Language:** Python 3
* **Libraries:** `numpy`, `pandas`, `matplotlib`, `seaborn`
* **Environment:** Jupyter Notebook
